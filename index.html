          • Building word usage examples pipelines for Wordnik.
            • Building pipelines that extract example sentences from various sources and gets them into online dictionary entries so that users can get an idea about how words are used.
              • One pipeline takes books dropped into storage, parses the epub format, breaks the contents up into sentences, filters those sentences, then bulk uploads them into Wordnik's database.
              • Another pipeline streams a very large (~20 GB) Wiktionary data dump using http range requests, uses hash checks against a Dynamo DB for changes in the Wiktionary entries from week to week, then extracts the updated entries' example sentences, and gets them into Wordnik's database.
              • A third pipeline uses a modified fork of the open source nyt-first-said to scrape the New York Times for words that it has never used before, then filters for certain conditions, extracts the surrounding sentences from the article, and puts them into Wordnik.
            • The pipelines are composed of AWS lambdas, chained via schedulers, direct invocation, or S3 triggers. Some are in Node and some are in Python (for ease of access to NLP libraries).
            • I build a custom observability scheme so that the progress of a source through the pipelines can be observed.
